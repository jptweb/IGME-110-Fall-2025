# IGME-110 Fall 2025 - Infographic Directory

This directory contains all student infographics organized by app proposal. Students created infographics based on app proposals developed earlier in the semester.

- Infographics are organized by the app proposal they showcase
- Each entry includes the student name, section, and a link to view the infographic
- Use this as inspiration for Project 3 (Prototyping)

  - **Section 08** = TTh 8:00AM-9:15AM
- Professor Jeffersons Sections: 4, 5, 6, 9 (note: I don't have names for these but they are added)
